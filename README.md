# 🚀 CF-Link-Proxy

### 让文件下载更快、更稳定的免费工具



</div>

## 📖 这是什么？

**CF-Link-Proxy** 是一个免费的文件下载加速工具，利用 Cloudflare 全球网络帮您下载那些速度慢或不稳定的文件。

> 💡 **简单来说**：就像给您的下载请求找了个“快递中转站”，让文件更快、更稳地到达您的电脑。

---

## ✨ 功能亮点

| 功能 | 说明 |
|------|------|
| ⚡️ **极速加载** | 页面瞬间打开，无需等待 |
| 🎨 **美观界面** | 支持白天/夜间模式，保护您的眼睛 |
| 📱 **手机扫码** | 生成二维码，手机也能用 |
| 🔄 **智能重置** | 下载完成后自动恢复，不用刷新页面 |
| ❌ **一键清除** | 一键清空链接，还能解决页面卡住问题 |
| 🌐 **下载加速** | 通过 Cloudflare 全球网络中转，提升下载速度 |
| 🔒 **安全可靠** | 无需注册，不保存您的任何数据 |

---

## 🎯 适用场景

✅ 下载国外软件/文件速度慢  
✅ 某些资源经常下载中断  
✅ 需要稳定下载大文件  
❌ **不适合**：浏览网页、观看视频（专为下载优化）

---

## 🚀 如何使用

### 方法一：使用在线版本（推荐）

1. 打开部署好的网站链接
2. 粘贴您要下载的文件链接
3. 点击“访问”按钮
4. 开始下载！

### 方法二：自己部署（免费）

**步骤 1：准备代码**
```
下载本项目 → 找到 functions/[[path]].js
```

**步骤 2：部署到 Cloudflare**
1. 登录 [Cloudflare](https://www.cloudflare.com/)
2. 进入 "Workers & Pages"
3. 创建 Pages 项目，上传代码
4. 完成！无需任何配置

> 💡 **全程免费**，Cloudflare 提供免费套餐

## 🔧 高级玩法（可选）

如果您懂一些技术，可以自定义请求头：

```javascript
// 在 functions/[[path]].js 中找到 specialCases
const specialCases = {
  "*": {  // 对所有网站生效
    "Origin": "DELETE",
    "Referer": "DELETE"
  },
  "download.example.com": {  // 对特定网站
    "Range": "KEEP"  // 支持断点续传
  }
};
```
