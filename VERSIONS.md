# 版本更新记录

## v2.0 (当前版本)

### 新增功能
- **🎨 夜间模式支持**
  - 自动检测系统主题偏好
  - 手动切换主题（Light/Dark Mode）
  - 通过 Cookie 记忆用户主题偏好，无闪烁加载
  
- **📱 二维码生成**
  - 为当前访问链接生成二维码
  - 支持手机扫码访问
  - 包含完整的代理链接信息
  
- **🎨 界面样式优化**
  - 动态渐变背景动画
  - 现代化卡片式输入框设计
  - 毛玻璃效果（Backdrop Blur）
  - 响应式设计，适配移动端
  - 流畅的动画过渡效果

- **🔄 交互体验提升**
  - 一键清除和重置功能
  - 智能状态重置（下载完成后自动恢复）
  - 键盘快捷键支持（`/` 键快速聚焦）
  - 加载状态可视化指示器

### 技术优化
- 所有 CSS 和图标内联，零外部依赖
- 服务端渲染主题，消除样式闪烁
- 优化的错误处理和状态管理

## v1.0

### 核心功能
- **🌐 基础代理下载**
  - 通过 Cloudflare 网络中继请求
  - 支持 HTTP/HTTPS 协议
  - 自动为裸域名添加 `https://` 前缀
  
- **🔧 请求头自定义**
  - 通过 `specialCases` 配置自定义请求头
  - 支持删除或保留特定请求头
  - 支持为特定域名设置独立规则

- **🤝 跨域支持**
  - 自动添加 CORS 响应头
  - 支持预检请求（OPTIONS）

### 部署特性
- 单文件部署（`functions/[[path]].js`）
- Cloudflare Pages 原生支持
- 无需环境变量配置
- 极简部署流程
